# Infrastructure Monitoring using Prometheus and Grafana

## Project Overview
This project demonstrates monitoring system metrics using Prometheus and visualizing them using Grafana.

## Tools Used
- Prometheus
- Node Exporter
- Grafana
- Docker

## Architecture
Node Exporter → Prometheus → Grafana

## Setup Instructions
1. Install Docker
2. Run: docker compose up -d
3. Access:
   - Prometheus: http://localhost:9090
   - Grafana: http://localhost:3000

## Metrics Monitored
- CPU Usage
- Memory Usage
- System Metrics

## Conclusion
Successfully implemented a monitoring stack using Prometheus and Grafana.
